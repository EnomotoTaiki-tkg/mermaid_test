# 課題項目
## 流れ図
### 条件
- 開始と終了ノードをつける
- 条件分岐を組み込む
- 5ノード以上
- カッコいいほど高得点

## 解答
```mermaid
flowchart LR;
  A([開始]) --> B([/xへの入力/]);
  B --> C{x == 5}
　C -- true -->  D[[x --]]
  D --> B
  C -- false --> F[xは5です]；
　F --> E([終了])
```

 end
